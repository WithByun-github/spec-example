# 09 — AI Agent 자동화 (공격 시나리오 자동 테스트 + 탐지율 리포트)

Status: ready-for-agent

> 출처: `docs/PRD.md` (F7, 성공기준 ⑥), `docs/TASKS.md` (Phase 7). 데모 하이라이트 — 내재화 가설 실증.

## What to build

공격 시나리오를 자동 생성·일괄 실행하고 탐지율·오탐율을 자동 리포팅하는 파이프라인을 만든다. "AI Agent로 보안솔루션을 자체 구축·검증할 수 있다"는 가설을 시각적으로 증명하는 산출물이다. 선택적으로 미탐 공격을 룰 yaml 초안으로 자동 생성하는 데모까지 포함한다.

## Acceptance criteria

- [ ] `tests/attack_scenarios/` — 공격 시나리오 자동 생성 스크립트 (인젝션·탈옥·인코딩 우회·PII 포함 입력 등)
- [ ] 시나리오 일괄 실행 → 게이트웨이 검사 결과 수집
- [ ] 탐지율·오탐율 자동 리포트 출력 (콘솔/마크다운)
- [ ] (선택) 룰 작성 Agent: 미탐 공격 발견 → `rules/*.yaml` 초안 자동 생성 데모
- [ ] **검증**: 시나리오 일괄 실행 → 탐지율 리포트 출력
- [ ] 모든 테스트 데이터는 합성값 (실제 개인정보 금지)
